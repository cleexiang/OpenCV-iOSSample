

##本项目用于演示基于OpenCV的一些图像处理

####混合模式
* 变暗 Darken 
* 正片叠底 Multiply
* 颜色加深 ColorBurn (NG)
* 线性加深 LinearBurn | 减去 Subtract
* 变亮 Lighten
* 滤色 Screen
* 颜色减淡 ColorDodge
* 线性减淡 LinearDodge | 添加 Add
* 叠加 Overlay
* 柔光 SoftLight
* 强光 HardLight
* 亮光 VividLight
* 线性光 LinearLight
* 点光 PinLight
* 实色混合 HardMix
* 差值 Difference
* 排除 Exclusion
